## 八、核心功能流程

### 8.1 意图共享流程

```
用户点击游戏卡片 
    → 跳转到PlayPage 
    → aboutToAppear生命周期 
    → 读取shareIntent.json 
    → 调用insightIntent.shareIntent() 
    → 系统处理意图数据 
    → 小艺建议卡片展示
```

### 8.2 意图调用流程

```
用户点击小艺建议卡片 
    → 系统拉起应用 
    → IntentExecutorImpl.onExecuteInUIAbilityForegroundMode() 
    → eventHub.emit发送事件 
    → Index监听事件 
    → 跳转到对应游戏页面
```

### 8.3 冷启动处理

```
应用未运行时点击意图卡片 
    → EntryAbility.onCreate() 
    → 检测launchReason为INSIGHT_INTENT 
    → 通过LocalStorage传递entityId 
    → Index页面读取并跳转
```

## 十一、项目特点

1. **意图框架集成**: 完整演示了HarmonyOS意图框架的使用，包括意图共享和意图调用
2. **多语言支持**: 支持中英文切换
3. **冷热启动处理**: 完善处理了应用的冷启动和热启动场景
4. **模块化设计**: 代码结构清晰，工具类、常量、模型分离
5. **备份恢复**: 支持应用数据备份恢复功能
6. **规范配置**: 完整的代码检查、混淆、构建配置
